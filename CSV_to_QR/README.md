# CSV_to_QR

A Django project to generate QR codes from CSV uploads and manage user accounts.

## Features
- User registration and login
- Upload CSV files and generate QR codes
- View and manage QR code data

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run migrations:
   ```bash
   python manage.py migrate
   ```
3. Start the server:
   ```bash
   python manage.py runserver
   ```
